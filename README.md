# Voice-Studio
Voice Studio is a text-to-speech application that converts text into natural speech. It includes Play, Pause, Rewind, and Loop controls, text translation, image upload with OCR, save functionality, and clear input options. It offers a simple, fast, and user-friendly experience for everyday use.
